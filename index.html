HTML

CHEATSHEET

*This is a personal cheatsheet that was made as a coding practice and for personal reference. It is not a complete documentation about html elements and may be subjected to some errors and typos. For a more reliable resource, I suggest to visit official websites like Codecademy or MDN Web Docs.

STRUCTURE

Tag Name Description
<html> HTML Root Element The root element of an HTML document.
<head> Head Section Contains metadata, links to CSS, and title information.
<body> Body Section Contains the content of the HTML document.

TEXT FORMATTING

Tag Name Description
<h1> - <h6> Headings Define headings, with <h1> being the largest and <h6> the smallest.
<p> Paragraph Defines a paragraph.
<b> Bold Makes text bold.
<i> Italic Makes text italicized.
<a> Anchor Creates a hyperlink.

Attributes: href (URL), target (_blank for new tab).
<ul>/<li> Unordered List Defines a bullet list.
<ol>/<li> Ordered List Defines a numbered list.

TABLES

Tag Name Description
<table> Table Defines a table.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines a cell in a table.
<th> Table Header Defines a header cell in a table.
<tfoot> Table Foot The set of rows defining the footer in a table.
Common attributes for <td> and <th> colspan :Specifies how amny colummns the cell spans.
rowspan :Specifies how many rows the cell spans.
